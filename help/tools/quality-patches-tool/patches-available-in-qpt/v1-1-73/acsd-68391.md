# ACSD-68391: Category-related permissions were not correctly applied in Quick Order and Requisition Lists

The ACSD-68391 patch fixes the issue where category-related permissions were not correctly applied in Quick Order and Requisition Lists. This patch is available when the [[!DNL Quality Patches Tool (QPT)]](/help/tools/quality-patches-tool/quality-patches-tool-to-self-serve-quality-patches.md) 1.1.73 is installed. The patch ID is ACSD-68391. Please note that this issue is scheduled to be fixed in Adobe Commerce 2.4.9.

## Affected products and versions

**The patch is created for Adobe Commerce version:**

* Adobe Commerce (all deployment methods) 2.4.8

**Compatible with Adobe Commerce versions:**

* Adobe Commerce (all deployment methods) 2.4.8 - 2.4.8-p3

>[!NOTE]
>
>The patch might become applicable to other versions with new [!DNL Quality Patches Tool] releases. To check if the patch is compatible with your Adobe Commerce version, update the `magento/quality-patches` package to the latest version and check the compatibility on the [[!DNL Quality Patches Tool]: Search for patches page](https://experienceleague.adobe.com/tools/commerce-quality-patches/index.html). Use the patch ID as a search keyword to locate the patch.

## Issue

Issue where category-related permissions were not correctly applied in Quick Order and Requisition Lists.

<u>Steps to reproduce</u>:

1. Enable Quick order and Requisition List.
1. Go to Stores > Configuration > Catalog > Catalog.
1. Scroll to Category Permissions and set Enable to Yes.
1. Configure category permissions for a category with denied price visibility for "NOT LOGGED IN" customer group.
1. Open the Category as a guest user on the storefront and confirm prices are hidden.
1. Use Quick Order to search for a product assigned to this category.
1. Add the product to the Requisition List.

<u>Expected results</u>:

1. Prices should not be visible for products from the restricted categories in Quick Order and Requisition List views.

<u>Actual results</u>:

1. Prices are visible in Quick Order and Requisition List for products from restricted categories.
