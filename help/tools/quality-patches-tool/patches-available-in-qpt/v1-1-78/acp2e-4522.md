# ACP2E-4522: Intermittent duplicate key errors occur on the quote_coupons table when multiple cart merge or quote save requests run at the same time

The ACP2E-4522 patch fixes the issue where an intermittent duplicate key error occurs on the quote_coupons table when multiple cart merge or quote save requests run at the same time. This patch is available when the [[!DNL Quality Patches Tool (QPT)]](/help/tools/quality-patches-tool/quality-patches-tool-to-self-serve-quality-patches.md) 1.1.78 is installed. The patch ID is ACP2E-4522. Please note that this issue is scheduled to be fixed in Adobe Commerce 2.4.9.

## Affected products and versions

**The patch is created for Adobe Commerce version:**

* Adobe Commerce (all deployment methods) 2.4.7-p7

**Compatible with Adobe Commerce versions:**

* Adobe Commerce (all deployment methods) 2.4.7 - 2.4.8-p4

>[!NOTE]
>
>The patch might become applicable to other versions with new [!DNL Quality Patches Tool] releases. To check if the patch is compatible with your Adobe Commerce version, update the `magento/quality-patches` package to the latest version and check the compatibility on the [[!DNL Quality Patches Tool]: Search for patches page](https://experienceleague.adobe.com/tools/commerce-quality-patches/index.html). Use the patch ID as a search keyword to locate the patch.

## Issue

An intermittent duplicate key error occurs on the `quote_coupons` table when multiple cart merge or quote save requests run at the same time.

<u>Steps to reproduce</u>:

1. Enable the maximum number of coupons per order = 10 under the Admin sidebar at **[!UICONTROL Stores]** > **[!UICONTROL Settings]** > **[!UICONTROL Configuration]** > **[!UICONTROL Sales]** > **[!UICONTROL Multicoupon Settings]**. 
1. Create a Cart Price Rule under **[!UICONTROL Marketing]** > **[!UICONTROL Promotions]** >  **[!UICONTROL Cart Price Rules]** > **[!UICONTROL  Add New Rule]**:
    1. **[!UICONTROL Rule Name]**: TESTCOUPON2026
    1. **[!UICONTROL Active]**: **[!UICONTROL Yes]**; **[!UICONTROL Websites]**: **[!UICONTROL Main Website]**; **[!UICONTROL Customer Groups]**; Select all applicable groups 
    1.  **[!UICONTROL Coupon]**: **[!UICONTROL Specific Coupon]**; **[!UICONTROL Coupon Code]**: TESTCOUPON2026.
    1.  **[!UICONTROL Uses per Customer]**: leave blank.
    1.  **[!UICONTROL Conditions]**: No conditions added.
    1. **[!UICONTROL Actions]** > **[!UICONTROL Percent of product price discount]**; **[!UICONTROL Discount Amount]** = 10.
1. Add a product with a quantity of at least 7 to a guest cart using GraphQL, then apply the coupon.
1. Run six concurrent parallel GraphQL mergeCarts calls:

   ```
   mutation {
     mergeCarts(
       source_cart_id: "<GUEST_CART_ID>"
       destination_cart_id: "<DESTINATION_CART_ID>"
     ) {
       id
       items { id product { sku } }
       applied_coupons { code }
     }
   }
   ```

<u>Expected results</u>:

No error is returned by the GraphQL merge.


<u>Actual results</u>:

The mergeCarts GraphQL mutation returns an error.
Logs show that the GraphQL failure is caused by a duplicate key constraint violation on the quote_coupons table (QUOTE_COUPONS_QUOTE_ID_COUPON_CODE) when multiple merge or quote save requests execute concurrently.

```
tail -n 200 var/log/exception.log var/log/debug.log | grep -E "Duplicate entry|QUOTE_COUPONS_QUOTE_ID_COUPON_CODE"
```
