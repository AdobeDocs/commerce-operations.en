# ACP2E-4110: Bundle product prices with a **[!UICONTROL special price]** differ between PDP/PLP and cart in a non-default currency

The ACP2E-4110 patch fixes the issue where bundle products with a **[!UICONTROL special price]** display incorrect amounts on the [!UICONTROL Product Detail Page] (PDP) and the [!UICONTROL Product Listing Page] (PLP) in a non-default currency. This patch is available when the [[!DNL Quality Patches Tool (QPT)]](/help/tools/quality-patches-tool/quality-patches-tool-to-self-serve-quality-patches.md) 1.1.79 is installed. The patch ID is ACP2E-4110. Please note that this issue is scheduled to be fixed in Adobe Commerce 2.4.9.

## Affected products and versions

**The patch is created for Adobe Commerce version:**

* Adobe Commerce (all deployment methods) 2.4.8-p3

**Compatible with Adobe Commerce versions:**

* Adobe Commerce (all deployment methods) 2.4.7 - 2.4.8-p4

>[!NOTE]
>
>The patch might become applicable to other versions with new [!DNL Quality Patches Tool] releases. To check if the patch is compatible with your Adobe Commerce version, update the `magento/quality-patches` package to the latest version and check the compatibility on the [[!DNL Quality Patches Tool]: Search for patches page](https://experienceleague.adobe.com/tools/commerce-quality-patches/index.html). Use the patch ID as a search keyword to locate the patch.

## Issue

Bundle products with a **[!UICONTROL special price]** display incorrect amounts on the PDP and the PLP in a non-default currency.

<u>Steps to reproduce</u>:

1. Create a simple product. Set the **[!UICONTROL price]** = *$40.25*.
1. Create a bundle product.
1. Set the **[!UICONTROL special price]** for the bundle product = *97.5%* and the **[!UICONTROL price view]** = *Price Range*.
1. Assign the simple product as an option of the bundle product. Set the **[!UICONTROL default quantity]** = *8*.
1. Set the **[!UICONTROL base currency]** = *GBP* for the main website.
1. Set the **[!UICONTROL currency conversion rate]** for USD to GBP = *0.75*.
1. Reindex.
1. Visit the storefront, view the bundle product and note the price.
1. Add the bundle product to the cart.
1. Check the price in the cart and compare it to the price on the product page.

<u>Expected results</u>:

The prices on the product page and in the cart should be the same.

<u>Actual results</u>:

The prices differ between the product page and the cart.
Price on the product page = *235.52 GBP*.
Price in the cart = *235.44 GBP*.
