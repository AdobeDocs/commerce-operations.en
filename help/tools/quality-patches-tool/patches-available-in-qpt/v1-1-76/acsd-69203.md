# ACSD-69203: Products List widget returns incorrect results when multiple categories are used in the category condition

The ACSD-69203 patch fixes the issue where the Products List widget returned incorrect results when multiple categories were listed in the category condition. This patch is available when the [[!DNL Quality Patches Tool (QPT)]](/help/tools/quality-patches-tool/quality-patches-tool-to-self-serve-quality-patches.md) 1.1.76 is installed. The patch ID is ACSD-69203. Please note that this issue is scheduled to be fixed in Adobe Commerce 2.4.9.

## Affected products and versions

**The patch is created for Adobe Commerce version:**

* Adobe Commerce (all deployment methods) 2.4.8-p3

**Compatible with Adobe Commerce versions:**

* Adobe Commerce (all deployment methods) 2.4.8 - 2.4.8-p3

>[!NOTE]
>
>The patch might become applicable to other versions with new [!DNL Quality Patches Tool] releases. To check if the patch is compatible with your Adobe Commerce version, update the `magento/quality-patches` package to the latest version and check the compatibility on the [[!DNL Quality Patches Tool]: Search for patches page](https://experienceleague.adobe.com/tools/commerce-quality-patches/index.html). Use the patch ID as a search keyword to locate the patch.

## Issue

The Products List widget returns incorrect results when multiple categories are specified in the category condition, displaying only a subset of products instead of all products assigned to the selected categories.

<u>Steps to reproduce</u>:

1.Log in as an admin.
1. Go to **[!UICONTROL Catalog]** > **[!UICONTROL Categories]** and create the following category structure:
    1. CAT1
        1. CAT11
        1. CAT12
    1. CAT2
        1. CAT21
        1. CAT22
1. For each category, set **[!UICONTROL Anchor]** to Yes and save.
1. Go to **[!UICONTROL Catalog]** > **[!UICONTROL Products]** and create four simple products (simple1, simple2, simple3, simple4).
1. Go to **[!UICONTROL Content]** > **[!UICONTROL Blocks]** and click **[!UICONTROL Add New Block]**.
    1. Set the block title to *home-page-block*.
    1. Click [!UICONTROL Edit with Page Builder].
1. Add a **[!UICONTROL Row]**, then drag an **[!UICONTROL HTML Code]** element into the row.
1. Click the gear icon of the **[!UICONTROL HTML Code]** element and click **[!UICONTROL Insert Widget]**.
1. Select **[!UICONTROL Catalog Products List]** as the **[!UICONTROL Widget Type]**.
1. Set conditions for categories CAT1 and CAT2:

```
Category is one of  3,4
```

1. Click **[!UICONTROL Insert Widget]** and save the block.
1. Go to **[!UICONTROL Content]** > **[!UICONTROL Pages]** and open the Home page.
1. Click **[!UICONTROL Edit with Page Builder]**.
1. Click **[!UICONTROL Add Content]** > **[!UICONTROL Block]** and select the home-page-block.
1. Save the Home page.
1. Load the home page from the frontend and check the loaded products.

<u>Expected results</u>:

All four products from all four categories are displayed.

<u>Actual results</u>:

Only two products from CAT11 and CAT12 are displayed.
