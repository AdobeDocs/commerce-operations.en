# ACSD-69325: Modifying the SKU case caused the product to appear out-of-stock on the storefront

The ACSD-69325 patch fixes the issue where modifying the SKU with lower case letters caused the product to be displayed out-of-stock on the storefront. This patch is available when the [[!DNL Quality Patches Tool (QPT)]](/help/tools/quality-patches-tool/quality-patches-tool-to-self-serve-quality-patches.md) 1.1.76 is installed. The patch ID is ACSD-69325. Please note that this issue is scheduled to be fixed in Adobe Commerce 2.4.9.

## Affected products and versions

**The patch is created for Adobe Commerce version:**

* Adobe Commerce (all deployment methods) 2.4.8-p3

**Compatible with Adobe Commerce versions:**

* Adobe Commerce (all deployment methods) 2.4.7 - 2.4.8-p3

>[!NOTE]
>
>The patch might become applicable to other versions with new [!DNL Quality Patches Tool] releases. To check if the patch is compatible with your Adobe Commerce version, update the `magento/quality-patches` package to the latest version and check the compatibility on the [[!DNL Quality Patches Tool]: Search for patches page](https://experienceleague.adobe.com/tools/commerce-quality-patches/index.html). Use the patch ID as a search keyword to locate the patch.

## Issue

Issue where modifying the SKU with lower case letters caused the product to be displayed out-of-stock on the storefront.

<u>Steps to reproduce</u>:

1. Log in to the Admin Panel.
1. Create additional website, store and store view.
1. Create sales channel for the new website.
1. Create a product assigned to the new website with SKU = 'RF-42R-80X100-CL', and assign it to the sales channel.
1. Run the indexer and cron.
1. Open the product page on the storefront.
1. Verify that product is **IN STOCK**.
1. Update the SKU as 'RF-42R-80x100-CL' (with lower case x).
1. Run the indexer and cron.
1. Open the product page on storefront
1. Verify the product stock status.

<u>Expected results</u>:

The product is shown as **IN STOCK**.

<u>Actual results</u>:

The product is shown as **Out of stock**.
