# Maintenance phase

The maintenance phase includes the following activities:

- Bug fixing
- Catalog management
- Configuration
- Feature enhancements
- Indexing
- Managed services
- Site monitoring
- Upgrades

The following sections include best practice information for the maintenance phase.
