# Extended support policy note

>[!NOTE]
>
>**Extended support** security patches are available to Adobe Commerce customers only. They are not available for the Magento Open Source code base.
>
>**Adobe Commerce on Cloud (release lines 2.4.4–2.4.6)** — Upgrade to a supported release or migrate to [!DNL Adobe Commerce as a Cloud Service] before your version upgrade enforcement date. Transitional support provisions do not replace Cloud upgrade enforcement.
>
>See also:
>
>- [Software lifecycle policy](https://experienceleague.adobe.com/en/docs/commerce-operations/release/planning/lifecycle-policy) — end-of-support dates, extended support, and security-only transitional provisions
>- [Cloud version upgrade enforcement policy](https://experienceleague.adobe.com/en/docs/commerce-operations/release/planning/version-upgrade-enforcement-policy) — enforcement dates and required actions
